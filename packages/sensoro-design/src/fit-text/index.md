## API

### FitText

| 参数         | 说明                 | 类型        | 默认值                    |
| ----------- | --------------------| ---------- | -------------------------|
| compressor  | 按比例缩小或增加字体大小 | number     | 1                        |
| minFontSize | 最小的字体大小         | number     | Number.NEGATIVE_INFINITY |
| maxFontSize | 最大的字体大小         | string     | Number.POSITIVE_INFINITY |
